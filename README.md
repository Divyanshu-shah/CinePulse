### Project Overview
The CinePulse project consists of multiple HTML files, each representing a different page on the website. The `index.html` file serves as the entry point, while other files like `about.html`, `movie.html`, and `event.html` provide specific content and functionality. The website's design system is defined in the `global.css` file, which imports Google Fonts and defines CSS custom properties. Additional styles are provided in separate CSS files, such as `home.css`, `movie.css`, and `event.css`.

The JavaScript code is organized into two main files: `book.js` and `main.js`. The `book.js` file handles the booking form logic, while the `main.js` file initializes various components, including the navbar, scroll reveal, and back-to-top functionality.
